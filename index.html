Orion Run

Um runner casual controlado por gestos de mao. Levante a mao para pular e feche a mao para agachar. Dificuldade leve para testar reconhecimento e fluidez.

Mao aberta = Pular
Mao fechada = Agachar
Dica: para usar a webcam, execute o jogo em um servidor local (ex.: http://localhost).
